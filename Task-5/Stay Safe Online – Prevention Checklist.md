# Stay Safe Online – Prevention Checklist

## Your Personal Guide to Digital Safety

---

## 📱 Device Security

| ✅ | Action | Frequency |
|---|--------|-----------|
| ☐ | Install antivirus/anti-malware software | Once |
| ☐ | Keep operating system and apps updated | Auto / Weekly |
| ☐ | Enable automatic security updates | Once |
| ☐ | Use a strong screen lock (PIN, pattern, fingerprint, or face unlock) | Always |
| ☐ | Avoid jailbreaking or rooting your device | Never |
| ☐ | Download apps only from official app stores (Google Play, Apple App Store) | Always |
| ☐ | Review app permissions – deny unnecessary access to contacts, SMS, location | Per app |
| ☐ | Disable Bluetooth and Wi-Fi when not in use | Always |
| ☐ | Enable "Find My Device" or similar tracking feature | Once |

---

## 🔐 Password & Authentication

| ✅ | Action |
|---|--------|
| ☐ | Use unique passwords for every account |
| ☐ | Create strong passwords – at least 12 characters with mix of uppercase, lowercase, numbers, and symbols |
| ☐ | Avoid using personal information (name, birthdate, etc.) in passwords |
| ☐ | Enable Two-Factor Authentication (2FA) on all important accounts (email, banking, social media) |
| ☐ | Use a password manager to store and generate passwords |
| ☐ | Never share passwords with anyone |
| ☐ | Change passwords immediately if you suspect a breach |

---

## 💳 UPI & Online Payments

| ✅ | Action |
|---|--------|
| ☐ | Never share UPI PIN, OTP, or debit/credit card details with anyone |
| ☐ | Set a daily transaction limit on UPI and cards |
| ☐ | Scan QR codes only from trusted sources |
| ☐ | Never install screen-sharing apps (AnyDesk, TeamViewer) at a stranger's request |
| ☐ | Verify payment recipients before transferring money |
| ☐ | Enable transaction alerts via SMS and email |
| ☐ | Check bank statements regularly for unauthorized transactions |
| ☐ | Remove old or unused UPI IDs and bank accounts from payment apps |

---

## 📧 Email & Messaging

| ✅ | Action |
|---|--------|
| ☐ | Never click on links or download attachments from unknown senders |
| ☐ | Verify the sender's email address – look for subtle misspellings |
| ☐ | Do not respond to emails asking for personal information |
| ☐ | Be skeptical of urgent messages claiming "account blocked" or "KYC update" |
| ☐ | Avoid forwarding unverified messages or chain messages |
| ☐ | Report spam and phishing emails to your email provider |

---

## 📞 Calls & Customer Support

| ✅ | Action |
|---|--------|
| ☐ | Never share OTP, UPI PIN, or bank details over phone calls |
| ☐ | Use only official customer support numbers from bank statements or official websites |
| ☐ | Do not trust search engine results for customer care numbers – scammers buy ads |
| ☐ | Hang up if someone claims to be from "bank technical team" asking for remote access |
| ☐ | Remember: Banks never ask for UPI PIN or OTP over call |

---

## 🌐 Internet Browsing

| ✅ | Action |
|---|--------|
| ☐ | Look for "https://" and padlock icon in address bar before entering sensitive information |
| ☐ | Avoid using public Wi-Fi for banking or sensitive transactions |
| ☐ | Use a VPN when connecting to public networks |
| ☐ | Clear browser cache and cookies regularly |
| ☐ | Enable "Do Not Track" or privacy settings in browser |
| ☐ | Use ad-blockers to avoid malicious advertisements |

---

## 📱 Social Media Safety

| ✅ | Action |
|---|--------|
| ☐ | Review and tighten privacy settings – limit who can see your posts and personal information |
| ☐ | Do not accept friend requests from strangers |
| ☐ | Avoid sharing live location, travel plans, or daily routines publicly |
| ☐ | Be cautious about quizzes and games asking for personal information |
| ☐ | Think before posting – content can be screenshotted and shared beyond your control |
| ☐ | Log out from shared devices |

---

## 🔄 Backup & Recovery

| ✅ | Action | Frequency |
|---|--------|-----------|
| ☐ | Back up important data to cloud or external drive | Weekly / Monthly |
| ☐ | Keep a secure record of passwords (use password manager) | Once |
| ☐ | Note down banking helplines and cybercrime reporting numbers | Once |
| ☐ | Save emergency contacts in your phone | Once |

---

## 🚨 If You Suspect Fraud – Act Fast

| ✅ | Immediate Action |
|---|------------------|
| ☐ | Call **1930** – National Cybercrime Helpline |
| ☐ | Report at **cybercrime.gov.in** |
| ☐ | Contact your bank immediately to block cards and UPI IDs |
| ☐ | Change passwords for compromised accounts |
| ☐ | Save evidence – screenshots, transaction IDs, call logs |
| ☐ | Visit local cyber police station if amount is significant |

---

## 📞 Important Helplines

| Service | Contact |

| NPCI UPI Helpline | 1800-120-1740 |

---

## 📊 Quick Self-Assessment

Ask yourself these questions monthly:

| Question | Yes / No |
|----------|---------|
| Do I use different passwords for different accounts? | ☐ |
| Have I enabled 2FA on my email and bank accounts? | ☐ |
| Do I verify senders before clicking links? | ☐ |
| Have I shared my OTP or PIN with anyone recently? | ☐ |
| Do I check bank statements regularly? | ☐ |
| Are my apps and OS updated? | ☐ |
| Do I know the number 1930? | ☐ |

**If you answered "No" to any question – take action today!**
